# Mariam Portfolio Starter

A professional Next.js portfolio starter based on your CV.

## 1) Create a new project

Use the official Next.js starter with the App Router. The official `create-next-app` CLI is the standard way to scaffold a new Next.js app, and Vercel supports automatic Git-based deployments for GitHub repositories. Tailwind CSS also provides a current Next.js setup guide. citeturn895253search3turn895253search4turn895253search2

```bash
npx create-next-app@latest mariam-portfolio
cd mariam-portfolio
```

Choose:
- TypeScript: **Yes**
- ESLint: **Yes**
- Tailwind CSS: **Yes**
- `src/` directory: **No**
- App Router: **Yes**
- Import alias: **Yes**

## 2) Replace these files

Copy the files from this starter into your new project, replacing:
- `app/page.tsx`
- `app/globals.css`
- `app/layout.tsx`

## 3) Run locally

```bash
npm run dev
```

## 4) Push to GitHub

```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```
